Description

This is an older version of my portfolio. It was designed to have a similar look and responsiveness as this one. The main considerations were 1. it responds appropriately to viewport sizes, 2. the desktop version has a fixed navigation area on the left, 3. the mobile version has a navigation menu at the bottom, and 4. social media buttons live at the bottom of the screen.

Execution

The fixed navigation area on the left of the desktop version lives inside a <header> element. This element flips up into a simple icon/name header in the mobile version.
